# 🧠 NeuroScan AI - Autism Screening Platform

🤖 AI-powered web application for autism screening and neurodiversity awareness.

## ✨ Features
- 📝 12-question behavioral assessment
- 📊 Interactive charts (radar, bar, pie)
- 💬 AI chatbot for queries
- 📄 Export reports to Word
- 🌈 Cyberpunk UI design

## 🛠️ Tech Stack
- 🔧 HTML, CSS, JavaScript
- 📈 Chart.js
- 🎨 Tailwind CSS
- 🤖 Google Gemini AI

## 📖 How to Use
1️⃣ Open home.html in browser  
2️⃣ Login or Signup  
3️⃣ Complete profile  
4️⃣ Take assessment  
5️⃣ View results  
6️⃣ Download report
